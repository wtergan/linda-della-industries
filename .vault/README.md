# Linda Della Industries engineering vault

This directory stores durable project planning, research, decisions, verification evidence, and goal handoffs. It is not a session transcript.

- `PLAN.md` — current roadmap and evidence index
- `plans/` — executable feature plans
- `research/` — source-backed market and product research
- `decisions/` — durable product and technical choices
- `goals/` — long-running goal state
- `visuals/` — screenshots and visual validation artifacts
