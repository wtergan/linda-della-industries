# Linda Della Industries

Launch site for a Baltimore-based AI systems implementation studio operated by William Doran IV.

## Current offer
A fixed-scope 7-Day AI Workflow Sprint for one owner-led business workflow. The launch price is $1,000 for founding clients, subject to fit and availability.

## Stack
- Astro 7 + TypeScript
- Original CSS/SVG identity system
- FormSubmit contact delivery with direct-email fallback
- Playwright + axe smoke/accessibility checks
- GitHub Pages deployment

## Local development
```bash
npm install
npm run dev
```

## Verification
```bash
npm run check
npm run build
npm run test:e2e
```

## Deployment
The workflow in `.github/workflows/deploy-pages.yml` builds and deploys `dist/` when `main` is pushed. GitHub Pages must use **GitHub Actions** as its source.

## Truth boundary
This site intentionally contains no customer logos, testimonials, case studies, compliance claims, or guaranteed ROI. Those may be added only after there is public, reviewable evidence and permission.
